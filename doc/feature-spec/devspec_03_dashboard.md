## 5. ユーザー操作と挙動

| 操作 | 挙動 |
|------|------|
| シグナルカードをクリック | `/trades` へ遷移（該当銘柄の履歴・詳細を確認） |
| ウォッチリスト銘柄をクリック | `/watchlist` へ遷移 |
| ログアウトボタン押下 | `POST /api/auth/logout` → `/login` へリダイレクト |

## 8. 実装メモ

```typescript
// app/page.tsx（RSCでデータ取得）
export default async function DashboardPage() {
  const [summary, signals, watchlist] = await Promise.all([
    fetch(`${API_BASE_URL}/api/portfolio/summary`, { cache: 'no-store' }),
    fetch(`${API_BASE_URL}/api/analysis/latest`,   { cache: 'no-store' }),
    fetch(`${API_BASE_URL}/api/watchlist`,          { cache: 'no-store' }),
  ]).then(res => Promise.all(res.map(r => r.json())));

  return (
    <>
      <SummaryCards summary={summary} />
      <LatestSignalList signals={signals} />
      <WatchlistSummary watchlist={watchlist} />
    </>
  );
}
```
