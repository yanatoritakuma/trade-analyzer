# 画面devspec - ウォッチリスト

**画面ID**: SCR-04  
**パス**: `/watchlist`（閲覧：全ユーザー） / `/admin/watchlist`（管理：admin のみ）  
**作成日**: 2026-06-06  
**バージョン**: 2.0

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 目的 | 全ユーザー共通のウォッチリストを表示・管理する |
| 閲覧 | user / admin（全員が同じウォッチリストを参照） |
| 追加・削除 | **admin のみ**（Claude APIコスト最適化のため） |
| 認証要否 | 必要 |
| 最大銘柄数 | 3銘柄（全ユーザー共通） |

## 4. 使用API

| メソッド | エンドポイント | 権限 | タイミング |
|---------|-------------|------|---------|
| GET | `/api/watchlist` | 全員 | 画面表示時 |
| POST | `/api/admin/watchlist` | admin のみ | 追加ダイアログで「追加する」押下時 |
| DELETE | `/api/admin/watchlist/:id` | admin のみ | 「削除」ボタン押下時 |

---

## 5. ユーザー操作と挙動

### 一般ユーザー
| 操作 | 挙動 |
|------|------|
| 画面表示 | 全ユーザー共通のウォッチリストを閲覧のみ |
| 追加・削除ボタン | 表示しない |

### 管理者
| 操作 | 挙動 |
|------|------|
| 「+ 銘柄を追加」押下 | 追加ダイアログを表示 |
| ダイアログで「追加する」押下 | API呼び出し → 一覧を再取得 |
| 「削除」ボタン押下 | 確認ダイアログ → DELETE |

---

## 6. バリデーション（追加ダイアログ・管理者のみ）

| フィールド | ルール | エラーメッセージ |
|-----------|--------|--------------|
| 銘柄コード | 必須 | 「銘柄コードを入力してください」 |
| 銘柄コード | 数字4桁 | 「銘柄コードは4桁の数字で入力してください」 |
| モード | 必須 | 「モードを選択してください」 |
| 銘柄数上限 | 3銘柄未満 | 「最大3銘柄まで登録できます」 |
| 重複チェック | 未登録 | 「すでに登録されている銘柄です」 |
