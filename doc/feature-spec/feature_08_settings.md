## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 目的 | プロフィール・実運用保有株の管理（書き込み操作のため管理者専用） |
| 対象ユーザー | **admin のみ**（一般ユーザーは閲覧のみのため本画面にアクセス不可） |
| 認証要否 | 必要（admin権限） |

## 3. 使用API

| メソッド | エンドポイント | タイミング |
|---------|-------------|---------|
| GET | `/api/auth/me` | 画面表示時（プロフィール取得） |
| PATCH | `/api/admin/me` | プロフィール保存時 |
| PUT | `/api/admin/me/password` | パスワード変更時 |
| GET | `/api/positions` | 保有株一覧取得（画面表示時） |
| POST | `/api/admin/positions` | 保有株追加時 |
| PUT | `/api/admin/positions/:id` | 保有株編集時 |
| DELETE | `/api/admin/positions/:id` | 保有株削除時 |

---

## 4. 各セクションの仕様

### プロフィール

| フィールド | 編集可否 | バリデーション |
|-----------|---------|-------------|
| お名前 | ✅ 可 | 必須・最大50文字 |
| メールアドレス | ❌ 不可（表示のみ） | - |

### パスワード変更

| フィールド | バリデーション |
|-----------|-------------|
| 現在のパスワード | 必須 |
| 新しいパスワード | 必須・8文字以上・英数字含む |
| パスワード確認 | 新しいパスワードと一致 |

### 実運用 保有株管理

#### 追加・編集ダイアログ

```
┌──────────────────────────────┐
│  保有株を追加 / 編集           │
│                              │
│  銘柄コード: [input]          │
│  取得単価:   [input]          │
│  保有数量:   [input]          │
│                              │
│  [キャンセル]  [保存する]      │
└──────────────────────────────┘
```

| フィールド | バリデーション |
|-----------|-------------|
| 銘柄コード | 必須・数字4桁 |
| 取得単価 | 必須・1以上の数値 |
| 保有数量 | 必須・1以上の整数 |

---

## 5. ユーザー操作と挙動

| 操作 | 挙動 |
|------|------|
| プロフィール「保存する」 | API呼び出し → 成功メッセージ表示 |
| パスワード「変更する」 | API呼び出し → 成功メッセージ → フォームをリセット |
| 「+ 保有株を追加」 | 追加ダイアログを表示 |
| 「編集」ボタン | 編集ダイアログを表示（既存値をプリセット） |
| 「削除」ボタン | 確認ダイアログ → 削除API呼び出し |

---

## 6. ローディング・エラー表示

| 状態 | UIの変化 |
|------|---------|
| 保存成功 | 「保存しました」のSnackbar（緑）を表示 |
| 保存失敗 | 「保存に失敗しました」のSnackbar（赤）を表示 |
| 現在のパスワード誤り | 「現在のパスワードが正しくありません」 |
