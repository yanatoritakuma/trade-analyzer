# 画面devspec - 週次レポート

**画面ID**: SCR-07  
**パス**: `/reports` / `/reports/:week`  
**作成日**: 2026-06-06  
**バージョン**: 2.0

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 目的 | AIが生成した週次学習レポートを一覧・詳細表示 |
| 対象ユーザー | user / admin（**閲覧のみ**・全ユーザー共通のレポートを閲覧） |
| 認証要否 | 必要 |
| データ | **管理者のtradesをもとに生成した全ユーザー共通のレポート** |

## 3. 使用API

| メソッド | エンドポイント | 用途 |
|---------|-------------|------|
| GET | `/api/reports` | レポート一覧取得 |
| GET | `/api/reports/:week` | レポート詳細取得（weekはYYYY-MM-DD形式） |

---

---

## 5. ユーザー操作と挙動

| 操作 | 挙動 |
|------|------|
| 「詳細を見る」押下 | `/reports/:week` へ遷移 |
| 「← 週次レポート一覧」押下 | `/reports` へ戻る |

---

## 6. ローディング・エラー表示

| 状態 | UIの変化 |
|------|---------|
| データ取得中 | カード・テキストをSkeleton表示 |
| レポートなし | 「まだレポートがありません。毎週日曜18:00に自動生成されます」 |
| API取得エラー | 「データの取得に失敗しました」 |
