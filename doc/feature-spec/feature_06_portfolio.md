## 3. 使用API

| メソッド | エンドポイント | 用途 |
|---------|-------------|------|
| GET | `/api/portfolio/summary` | 損益サマリー取得 |
| GET | `/api/positions` | 実運用保有株取得 |
| GET | `/api/trades?mode=virtual` | バーチャル保有ポジション取得 |

---

## 4. 表示データ仕様

### 損益グラフ

| 項目 | 内容 |
|------|------|
| グラフ種類 | 折れ線グラフ（Recharts LineChart） |
| X軸 | 日付 |
| Y軸 | 累計損益（円） |
| 期間切り替え | 1ヶ月 / 3ヶ月 / 6ヶ月 |

### 保有ポジション

| 列 | 内容 |
|----|------|
| 銘柄 | `6522 アステリスク` |
| 取得単価 | `¥1,050` |
| 数量 | `100株` |
| 現在値 | `¥1,180` |
| 含み益/損 | `+¥13,000`（プラス緑・マイナス赤） |
| 損益率 | `+12.4%` |

---

## 5. ユーザー操作と挙動

| 操作 | 挙動 |
|------|------|
| タブ切り替え | バーチャル / 実運用を切り替え |
| 期間ボタン切り替え | グラフの表示期間を変更 |
| ポジション行クリック | トレード履歴（その銘柄）へ遷移 |

---

## 6. ローディング・エラー表示

| 状態 | UIの変化 |
|------|---------|
| データ取得中 | グラフ・テーブルをSkeleton表示 |
| ポジションなし | 「現在保有中のポジションはありません」 |
| API取得エラー | 「データの取得に失敗しました」 |
