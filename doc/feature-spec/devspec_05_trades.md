## 3. 使用API

| メソッド | エンドポイント | タイミング |
|---------|-------------|---------|
| GET | `/api/trades?mode=virtual` | バーチャルタブ表示時 |
| GET | `/api/trades?mode=real` | 実運用タブ表示時 |

### クエリパラメータ

| パラメータ | 型 | 説明 |
|-----------|-----|------|
| mode | string | `virtual` / `real` |
| ticker | string | 銘柄コードで絞り込み |
| action | string | `BUY` / `SELL` |
| from | string | 開始日（YYYY-MM-DD） |
| to | string | 終了日（YYYY-MM-DD） |

---

## 4. 表示データ仕様

| 列 | 内容 |
|----|------|
| 日付 | `2026-06-01` |
| 銘柄 | `6522 アステリスク` |
| 売買 | 🟢 BUY / 🔴 SELL |
| 単価 | `¥1,180` |
| 数量 | `100株` |
| 損益 | `+¥4,200`（プラス緑・マイナス赤・未確定はグレー） |
| 信頼度 | `85%` |

---

## 5. ユーザー操作と挙動

| 操作 | 挙動 |
|------|------|
| タブ切り替え | バーチャル / 実運用を切り替えて再取得 |
| 銘柄フィルタ | 選択した銘柄のみ表示 |
| 期間フィルタ | 日付範囲で絞り込み |
| BUY/SELLフィルタ | アクションで絞り込み |
| 行クリック | トレード詳細モーダルを表示 |

### トレード詳細モーダル

## 6. ローディング・エラー表示

| 状態 | UIの変化 |
|------|---------|
| データ取得中 | テーブル全体をSkeleton表示 |
| データなし | 「トレード履歴がありません」 |
| API取得エラー | 「データの取得に失敗しました」 |
